# fract'ol
42 graphics project:
Display fractals using C Language and MinilibX

---
---

Run `make` to compile libraries and project

Usage: `./fractol [ 1 | 2 | 3 | 4 | help ]`
- 1: Mandelbrot set
- 2: Julia set
- 3: Burning ship
- 4: Julia (with absolute values)
